Etiquetas semánticas de estructura

Esta etiqueta contenedora no tiene semántica
header - Cabecera de un sitio web o una sección
main - Define la sección principal del documento(debe haber una sola etiqueta main en el documento)
footer - Pié de página de un sitio web o de una sección
nav - Representa una navegación
article - Representa una sección autocontenido(que por si sola se explica)
aside - Representa contenido complementario o secundario(por ejemplo, publicidad)
section - Representa una sección de contenido genérico
address - Representa una dirección física o información de contacto

Etiquetas de bloque VS etiquetas de linea

La etiqueta de bloque por excelencia es la div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas, ocupará todo el ancho de la pantalla
La etiqueta de linea por excelencia es el span, uan etiqueta de linea ocupa solo el espacio que tiene su contenido ☝️
